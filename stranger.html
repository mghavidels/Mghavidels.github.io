Light 3d painting

The idea of this project was fully conceptual.We had a story. A stranger that comes from another world for having communication to the human being.The stranger can travel to the time but it is not able to touch out materials.We can touch it too. So we decided to create a special space for having communication with the stranger.The stranger is a thing that we can just see it. we are not able to touch it.Since it is not like other material of our world we know it does not exist. Our challenge was how we can having communication with something that does not exist.we needed some tools . Kinect, Processing and projector.Processing is a code base software that the output of it is an interactive visual picture.How it works?When a user shake his hand the Kinect detect the motion. The information goes to the processing code that is connected to the kinect.according to the motion of user code changes.Projector project the interactive picture of processing code to the clear cube. Clear cube that was made of glass is filled of fog.so the picture will be like volume light.For having a 3d light we instal 2 projector ..
